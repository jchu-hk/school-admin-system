# PM进度检查定时任务

## 设置定时检查 (每小时)

### OpenClaw Cron 设置

在 OpenClaw 中设置每小时自动检查:

```
/cron add progress-check "0 * * * *" "检查项目进度并报告"
```

### 或使用外部定时器

```bash
# 每小时运行一次
0 * * * * cd /workspace/projects/workspace && bash scripts/auto-progress-check.sh >> logs/cron-progress.log 2>&1
```
