# 每日站会流程

每日站会（Daily Standup）用于同步团队进度，识别阻塞和风险。

---

## 基本信息

- **时间**：每日 09:00（或项目团队约定时间）
- **时长**：15分钟内
- **参会人员**：PM + DEV1/2/3 + QA1/2 + DEVOPS

---

## 会议流程

1. **昨日完成情况回顾**（每人 1-2 分钟）
2. **今日计划任务**（每人 1-2 分钟）
3. **阻塞问题和风险**（重点讨论）
4. **需要协助的事项**（明确指派）

## 注意事项

1. **控制时间**：每人发言不超过 2 分钟
2. **聚焦问题**：不展开技术讨论，站会后单独跟进
3. **明确行动**：阻塞问题必须明确负责人和时间
4. **及时记录**：会后立即更新站会记录
