## 1. 文档概述

### 1.1 目的

本界面原型设计文档用于向高级管理层 (Senior Management) 和部门主管 (Department Heads) 展示智能校务助理系统的用户界面设计，帮助他们全面了解系统的操作体验和功能呈现方式。

### 1.2 适用范围

本设计覆盖以下用户角色：

| 用户角色 | 说明 | 权限级别 |
|----------|------|----------|
| **校务主任** | 系统管理员，全面管理 | 最高权限 |
| **部门主管** | 各部门负责人 | 部门权限 |
| **教职员** | 教师、职工 | 基础权限 |
| **家长** | 学生家长查询 | 只读权限 |
| **AI 助理** | 智能问答 | 系统角色 |

### 1.3 模块与功能映射

基于 **SPEC-COMPLETE.md v1.3.0** 的功能模块：

| 模块编号 | 模块名称 | 功能数量 | 界面复杂度 |
|----------|----------|----------|------------|
| Module 1 | MOD-DAILY-001 每日仪表板 | 7 | ⭐⭐⭐ 高 |
| Module 2 | MOD-CYCL-001 周期性校务 | 10 | ⭐⭐⭐ 高 |
| Module 3 | MOD-FIN-001 财务资产 | 7 | ⭐⭐⭐ 高 |
| Module 4 | MOD-USER-001 用户管理 | 7 | ⭐⭐ 中 |
| Module 5 | MOD-AI-001 AI 智能助理 | 5 | ⭐⭐⭐ 高 |
| Module 6 | MOD-INT-001 系统集成 | 6 | ⭐ 基础 |
| Module 7 | (合规管理) | - | ⭐⭐ 中 |
| Module 8 | MOD-I18N-001 多语言 | 4 | ⭐ 基础 |

## 2. 设计原则与规范

### 2.1 设计系统

| 项目 | 设计值 |
|------|--------|
| **设计框架** | React 18 + Tailwind CSS |
| **图标库** | Lucide React |
| **字体** | 中文: Noto Sans HK / 英文: Inter |
| **基准字号** | 14px (正文), 16px (标题) |
| **栅格系统** | 12 列, 间距 16px / 24px / 32px |
| **圆角** | 4px (按钮), 8px (卡片), 12px (模态框) |

### 2.2 颜色系统

| 用途 | 颜色 | Hex |
|------|------|-----|
| **主色 (Primary)** | 深蓝色 | #1E40AF |
| **辅助色 (Secondary)** | 浅蓝色 | #3B82F6 |
| **成功 (Success)** | 绿色 | #10B981 |
| **警告 (Warning)** | 橙色 | #F59E0B |
| **错误 (Error)** | 红色 | #EF4444 |
| **背景色** | 浅灰白 | #F8FAFC |
| **文字色** | 深灰 | #1E293B |
| **边框色** | 浅灰 | #E2E8F0 |

### 2.4 组件库

| 组件类型 | 说明 | 使用场景 |
|----------|------|----------|
| **Button** | 主按钮、次按钮、危险按钮、文字按钮 | 操作入口 |
| **Input** | 文本输入、下拉选择、日期选择 | 数据录入 |
| **Table** | 数据表格，支持排序、筛选、分页 | 数据展示 |
| **Card** | 信息卡片、统计卡片 | 数据汇总 |
| **Modal** | 模态对话框 | 详情查看/编辑 |
| **Toast** | 成功、警告、错误提示 | 操作反馈 |
| **Badge** | 状态标签、计数徽章 | 状态标识 |
| **Avatar** | 用户头像、图标 | 用户标识 |
