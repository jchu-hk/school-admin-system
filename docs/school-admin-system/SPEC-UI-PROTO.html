📋 文档概述

📌 参考依据

本界面原型设计文档基于 SPEC-COMPLETE.md v1.3.0 功能规格书，包含 8 大模块、49 个功能函数。

目的

本界面原型设计文档用于向高级管理层 (Senior Management) 和部门主管 (Department Heads) 展示智能校务助理系统的用户界面设计，帮助他们全面了解系统的操作体验和功能呈现方式。

适用用户角色

用户角色 说明 权限级别
🏢 校务主任 系统管理员，全面管理 最高权限
👔 部门主管 各部门负责人 部门权限
👨‍🏫 教职员 教师、职工 基础权限
👪 家长 学生家长查询 只读权限
