### 2.1 登录认证流程

| 测试项 | 前端端点 | 后端端点 | 状态 | 问题描述 |
|--------|----------|----------|------|----------|
| 登录页面 | POST /api/auth/login | ❌ 未实现 | 🔴 失败 | 后端缺少Auth模块 |
| OTP验证 | POST /api/auth/verify-otp | ❌ 未实现 | 🔴 失败 | OTP验证接口缺失 |
| Token刷新 | POST /api/auth/refresh | ❌ 未实现 | 🔴 失败 | Token刷新机制缺失 |
| 权限验证 | - | ❌ 未实现 | 🔴 失败 | JWT Guard未配置完整 |

**问题详情**:
- **严重**: 后端完全没有 `AuthController` 和认证相关服务
- **严重**: `JwtAuthGuard` 存在但缺少登录接口生成token
- **中等**: OTP模块存在但缺少与登录流程的集成

**修复建议**:
```typescript
// 需要创建 apps/backend/src/modules/auth/auth.controller.ts
@Controller('api/auth')
export class AuthController {
  @Post('login')
  async login(@Body() loginDto: LoginDto) { ... }
  
  @Post('verify-otp')
  async verifyOtp(@Body() otpDto: VerifyOtpDto) { ... }
  
  @Post('refresh')
  async refreshToken(@Body() refreshDto: RefreshDto) { ... }
}
```
