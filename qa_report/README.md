## 🚀 快速开始

### 安装测试环境

```bash
# 1. 进入测试目录
cd e2e-tests

# 2. 安装依赖
pnpm install

# 3. 安装浏览器
pnpm exec playwright install

# 4. 运行冒烟测试
pnpm test:smoke

# 5. 查看报告
pnpm report
```

### 常用命令

```bash
# 运行全部测试
pnpm test

# 运行指定模块
pnpm test:attendance
pnpm test:finance

# 运行指定浏览器
pnpm test:chrome
pnpm test:firefox
pnpm test:webkit

# 调试模式
pnpm test:debug

# UI模式
pnpm test:ui
```
