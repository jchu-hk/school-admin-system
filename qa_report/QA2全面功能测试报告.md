## 📋 测试结果汇总表

### 1. 登录认证模块

| 测试项 | 结果 | 严重程度 | 备注 |
|--------|------|---------|------|
| 用户名密码登录 | ⚠️ 部分通过 | P1 | 登录成功但 Token 存储为 "undefined" |
| OTP验证 | ⏸️ 未测试 | - | 需要真实 OTP 配置 |
| Token刷新 | ⏸️ 无法测试 | P1 | Token 存储错误导致无法测试 |
| 登出功能 | ⏸️ 无法测试 | P2 | Token 存储错误导致侧边栏无法点击 |

**详情：**
- 登录表单渲染 ✅
- 表单验证（用户名非空、密码长度） ✅
- Mock 登录返回成功，跳转 Dashboard ✅
- **Bug：** `Login.tsx` 读取 `res.data.access_token`，但 Mock 返回 `data.data.token`，导致 `localStorage.token = "undefined"` ⚠️
- **Bug：** 后端返回 `accessToken`（驼峰），前端期望 `access_token`（下划线）⚠️
