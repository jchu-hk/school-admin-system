### 3. Key Decisions

- **Secrets Management:** Using HashiCorp Vault for centralized secrets management (P0 priority)
- **Password Hashing:** Migrating from bcrypt to Argon2id for better security
- **Database Architecture:** PostgreSQL Primary + 2 Replicas + PgBouncer for connection pooling
- **Redis Architecture:** Redis Cluster (6 nodes, 3M+3S) instead of Sentinel
- **Monitoring:** Three-pillar observability (Metrics + Logs + Traces)
- **DR Strategy:** Active-Passive failover with RTO 4h (critical services), RPO 15min (critical data)
