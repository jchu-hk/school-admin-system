## 1. 设计原则

### 1.1 核心设计理念

| 原则 | 说明 | 应用示例 |
|------|------|----------|
| **效率优先** | 高频操作路径≤3步，减少用户操作负担 | 一键审批、快捷回复模板 |
| **信息可见** | 关键数据一目了然，避免深层嵌套 | 仪表板统计卡片、实时状态指示 |
| **容错设计** | 操作可撤销，重要操作需确认 | 批量撤销、二次确认弹窗 |
| **一致性** | 相同功能保持相同交互模式 | 统一的表格操作、表单校验 |
| **情境感知** | 根据用户角色和场景智能适配 | 权限控制、数据范围自动过滤 |

### 1.2 设计系统

| 项目 | 设计值 |
|------|--------|
| **设计框架** | React 18 + Tailwind CSS |
| **图标库** | Lucide React |
| **字体** | 中文: Noto Sans HK / 英文: Inter |
| **基准字号** | 14px (正文), 16px (标题) |
| **栅格系统** | 12 列, 间距 16px / 24px / 32px |
| **圆角** | 4px (按钮), 8px (卡片), 12px (模态框) |

### 1.3 颜色系统

| 用途 | 颜色 | Hex | 使用场景 |
|------|------|-----|----------|
| **主色 (Primary)** | 深蓝色 | #1E40AF | 主要按钮、链接、高亮 |
| **辅助色 (Secondary)** | 浅蓝色 | #3B82F6 | 次级按钮、悬停状态 |

## 3. 全局组件规范

### 3.1 按钮规范

#### 按钮类型

| 类型 | 样式 | 使用场景 | 悬停效果 | 禁用状态 |
|------|------|----------|----------|----------|
| **主按钮** | 背景:#1E40AF, 文字:white | 主要操作（保存、提交） | 背景:#1E3A8A | opacity:0.5 |
| **次按钮** | 背景:#F1F5F9, 边框:#E2E8F0 | 次要操作（取消、返回） | 边框:#CBD5E1 | opacity:0.5 |
| **危险按钮** | 背景:#EF4444, 文字:white | 删除、拒绝操作 | 背景:#DC2626 | opacity:0.5 |
| **成功按钮** | 背景:#10B981, 文字:white | 批准、确认操作 | 背景:#059669 | opacity:0.5 |
| **文字按钮** | 无背景, 文字:#1E40AF | 链接、低优先级操作 | 文字:#1E3A8A | opacity:0.5 |
| **图标按钮** | 36×36px, 圆角8px | 表格操作、工具栏 | 背景:#F1F5F9 | opacity:0.5 |

#### 按钮尺寸

| 尺寸 | 高度 | 内边距 | 字号 | 使用场景 |
|------|------|--------|------|----------|
| **大按钮** | 44px | 12px 24px | 15px | 登录页、重要CTA |
| **默认按钮** | 36px | 8px 16px | 13.5px | 常规操作 |
| **小按钮** | 28px | 5px 11px | 12px | 表格内操作、紧凑布局 |

#### 按钮交互状态

```
默认状态 → 悬停状态 (background darken 10%)
        → 点击状态 (transform: scale(0.97))
        → 加载状态 (显示加载图标，禁用点击)
        → 成功状态 (显示✓图标，绿色背景)
        → 错误状态 (显示✗图标，红色背景)
```

### 3.2 表单校验规范

#### 校验时机

| 校验类型 | 触发时机 | 提示方式 |
|----------|----------|----------|
| **实时校验** | 输入完成且失焦 (blur) | 输入框下方红色文字 |
| **即时校验** | 输入过程中 (input) | 输入框边框变色 |
| **提交校验** | 点击提交按钮 | 全局提示 + 错误字段聚焦 |

#### 校验样式

| 状态 | 输入框边框 | 提示文字 | 图标 |
|------|------------|----------|------|
| **默认** | #E2E8F0 | 无 | 无 |
| **聚焦** | #3B82F6 + 阴影 | 无 | 无 |
| **错误** | #EF4444 | #EF4444, 12px | ⚠️ 图标 |
| **成功** | #10B981 | #10B981, 12px | ✓ 图标 |
| **警告** | #F59E0B | #F59E0B, 12px | ⚡ 图标 |

#### 常见校验规则

| 字段类型 | 校验规则 | 错误提示 |
|----------|----------|----------|
| **必填** | 不能为空 | "此字段为必填项" |
| **邮箱** | 符合邮箱格式 | "请输入有效的邮箱地址" |
| **手机号** | 8位香港手机号 | "请输入有效的手机号码" |
| **身份证** | 香港身份证格式 | "身份证格式不正确" |
| **日期** | 不能晚于今天 | "日期不能晚于今天" |
| **金额** | 正数，≤2位小数 | "请输入有效的金额" |
| **密码** | ≥8位，含大小写+数字 | "密码强度不足" |

### 3.3 表格交互规范

#### 表格结构

| 元素 | 样式 | 说明 |
|------|------|------|
| **表头** | 背景:#F8FAFC, 文字:#64748B, 12px | 可点击排序 |
| **行高** | 56px | 确保触控友好 |
| **行悬停** | 背景:#F8FAFC | 整行高亮 |
| **选中行** | 背景:#EFF6FF | 多选状态 |
| **斑马纹** | 交替背景色 | 可选开启 |

#### 表格操作

| 操作 | 交互方式 | 说明 |
|------|----------|------|
| **排序** | 点击表头，三角图标指示方向 | 支持多列排序 |
| **筛选** | 表头下拉或独立筛选栏 | 支持多条件组合 |
| **分页** | 底部分页器 | 默认20条/页 |
| **行选择** | 首列复选框 | 支持全选/取消 |
| **行操作** | 尾列操作按钮组 | 编辑、删除、查看 |
| **批量操作** | 选中后顶部显示操作栏 | 删除、导出、审批 |

#### 分页组件

```
[上一页] [1] [2] [3] ... [10] [下一页]    显示 1-20 条，共 186 条    [20条/页 ▼]
```

### 3.4 弹窗规范

#### 弹窗类型

| 类型 | 尺寸 | 用途 | 关闭方式 |
|------|------|------|----------|
| **确认弹窗** | 400×200px | 二次确认操作 | 确认/取消按钮 |
| **信息弹窗** | 500×auto | 详情展示、预览 | ×按钮、遮罩点击 |
| **表单弹窗** | 600×auto | 编辑、创建 | 保存/取消按钮 |
| **大弹窗** | 800×600px | 复杂操作、报表 | ×按钮 |
| **全屏弹窗** | 100%×100% | 数据同步、批量处理 | 完成按钮 |

#### 弹窗层级

| 层级 | z-index | 用途 |
|------|---------|------|
| **遮罩层** | 1000 | 背景半透明黑色 |
| **普通弹窗** | 1010 | 确认、信息 |
| **重要弹窗** | 1020 | 删除确认 |
| **Toast提示** | 1100 | 全局通知 |
| **加载遮罩** | 1200 | 页面加载中 |

#### 弹窗动画

```
打开: opacity 0→1, scale 0.95→1, duration 200ms, ease-out
关闭: opacity 1→0, scale 1→0.95, duration 150ms, ease-in
```
